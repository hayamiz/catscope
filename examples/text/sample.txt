This is a sample plain text file used for testing the file browser preview.
It contains multiple paragraphs, some indentation, and varied formatting to
exercise the text rendering pipeline.

Key features of the catscope file browser:
    1. Real-time file change notifications via WebSocket
    2. Embedded static assets served from memory
    3. Zero external dependencies for the frontend
    4. Path traversal protection on all endpoints

    Technical notes:
        - The server resolves symlinks before checking path boundaries.
        - EPS conversion is optional and degrades gracefully.
        - File watching uses fsnotify with 100ms debounce per file.
